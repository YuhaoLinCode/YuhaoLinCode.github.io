# Yuhao Lin — Personal Homepage

Personal academic homepage of Yuhao Lin (林宇浩).

Built as a static single-page site. Content is driven by two YAML files:

- `data/site.yml` — personal info, navigation, about, honors
- `data/publications.yml` — publication list with media, links, and author metadata
